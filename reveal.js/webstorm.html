Editing

WebStorm is really Smart for editing

  • Auto complete for html/css (ctrl+space)
  • Auto complete for imported JS library
  • Jump between methods
  • Auto format
  • Refactor

Advanced Editing

  • Live Template (ctrl J, ctrl alt T), Surround with (Emmet Support)
  • Multi carets/select (ctrl+shift+alt+click, double ctrl + arrow)
  • column selection mode (alt + mouse)
  • Paste from history / local history

Keyboard shortcuts

  • auto complete (ctrl + space)
  • comment (ctrl + /)
  • format (ctrl + alt + l)
  • previous caret/change (ctrl+alt left/right, ctrl+shift+backspace)
  • previous next function (alt + up/down)
  • Recent files / Recent changed files (ctrl+E, ctrl+shift+E)
